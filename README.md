# ✂️ Barber Shop — Frontend

Sitio web completo para una barbería moderna con sistema de reservas y panel de administración.

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 📄 Páginas

| Página | Descripción |
|--------|-------------|
| `home.html` | Página principal con hero, historia, ubicación y barberos |
| `reservas.html` | Formulario para que el cliente reserve su cita |
| `login.html` | Panel de acceso para administradores |
| `dashboard.html` | Panel de reservas para barberos con filtros |

## ✨ Funcionalidades

- Navegación con scroll suave entre secciones
- Formulario de reservas conectado al backend
- Carga dinámica de barberos y horarios desde la API
- Validaciones en el formulario antes de enviar
- Login de administrador con JWT
- Dashboard protegido, redirige al login si no hay sesión
- Filtro de reservas por fecha y barbero
- Visualización de total de reservas por barbero
